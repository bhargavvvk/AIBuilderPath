## Observation

While working on one long conversation in ChatGPT related to my major project documentation and development, I observed how the model handled larger context over time.

Initially, the responses were fast and accurate. The model was able to understand:
- project requirements
- research papers
- implementation details
- debugging discussions
- documentation drafting

As the conversation became larger and more detailed, I observed some issues:

- Response generation became slower.
- The model sometimes forgot earlier context.
- It occasionally mixed draft versions with finalized content.
- Some responses became inconsistent with previous instructions.
- Context switching between documentation, coding, and explanations caused confusion in certain responses.
